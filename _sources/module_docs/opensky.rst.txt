OpenSky package API
===================

This package handles loading and agglomerating `OpenSky <https://opensky-network.org/>`_ flight data. OpenSky already
has `a useful Python interface <https://github.com/xoolive/traffic>`_ so this package is focused first on providing
tools to read in the data made available by Strohmeier et al. in their `2020 ESSD paper <https://essd.copernicus.org/preprints/essd-2020-223/>`_.
