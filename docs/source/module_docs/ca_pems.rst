CA PEMS package API
===================

This package focuses on reading data from the `Caltrans PEMS website <http://pems.dot.ca.gov/>`_. Its primary focus is
on data available through the Data Clearinghouse. Currently, it has support for the "station" files only. Support for
other file types (FasTrak, weights, etc.) will be added as time and demand permit.
