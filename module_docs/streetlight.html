Streetlight package API¶

Streetlight made available vehicle-miles-traveled data for all US counties during the main COVID-19 pandemic in the US. This package includes functions to read this into a dataframe.

Module: readers¶

caada.streetlight.readers.load_streetlight_csv(filename: Union[str, pathlib.Path]) → geopandas.geodataframe.GeoDataFrame¶

Load the Streetlight VMT data from a .csv file as a Geodataframe with county geometry included

Parameters

filename – Path to the Streetlight VMT file. Note that at present it MUST be converted to a .csv file - directly loading the Excel file is not yet supported.

Returns

A geodataframe with the VMT data, joined with the county geometry

Return type

geopandas.GeoDataFrame
