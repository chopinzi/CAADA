EIA package API¶

This package focuses on reading in data from the US Energy Information Administration. The EIA has a large amount of data, but most of it is at fairly coarse time and space resolution, so this package focuses on the higher resolution data.

Module: readers¶

caada.eia.readers.load_eia_df(filename: Union[str, pathlib.Path], with_units: bool = False)¶

Load an EIA electricity consumption chart CSV as a dataframe

Parameters

  • filename – A path to a .csv file that is the CHART download from https://www.eia.gov/electricity/data/browser/#/topic/.

  • with_units – If True, then a dictionary of units is returned alongside the dataframe. If False, only the dataframe is returned.

Returns

  • pandas.DataFrame – A dataframe corresponding to the .csv file. The index will be a datetime index matching the times in the .csv file and the columns will be a multiindex with the first level giving the region (e.g. United States, California) and the second level giving the sector.
